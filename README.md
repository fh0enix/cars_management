# How to install and run project?

## Firs of all you need to:

### 1. Download progect

```bash
git clone https://github.com/fh0enix/cars_management
```

### 2. Enter in progect folder

```bash
cd <progect folder>
```

### 3. Run gem Bundler  in terminal

```bash
bundler install
```

### 4. Run my app in terminal

```bash
 ruby index.rb
```

## 5. Enjoy)
